# 世界之窗天文钟模拟器

本项目将天文钟模拟器的默认时区及经纬度修改为深圳市，并在历史记录中保留布拉格天文钟的时区及经纬度，方便快速切换查看。并将web界面及选项汉化为简体中文。

仅为个人使用，无任何盈利行为。

深圳世界之窗天文钟按布拉格天文钟的原理复刻并建造，钟面按照深圳世界之窗所在地的实际纬度重新划分，以指示正确的日出/日落时间等天文现象。

原项目地址：[kshetline/prague-clock](https://github.com/kshetline/prague-clock)

以下为原项目说明（翻译为中文）：

这是对世界著名的布拉格天文钟（Pražský Orloj）的模拟，这是一座中世纪天文钟，最早于 1410 年安装，至今仍在运行（期间经历过损坏和多次修复）。

真实世界的钟表是为布拉格的特定经纬度设计的，大约在北纬50.1°，东经14.4°。然而，这个模拟器可以设置为地球上的任何位置，你可以显示当前时间，或者设置一个特定时间来查看该时间下天文钟将如何显示。

模拟器的指针、外环时间、太阳、月亮和黄道盘的位置默认使用您的计算机时钟和现代天文计算来确定。也可以选择近似模拟机械计时。

此处使用的原始天文钟 SVG 图形和字体最初由 Jan Tošovský 开发，可以在 [该网站](https://drifted.in/horologium-app/) 找到。
